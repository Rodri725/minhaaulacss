Sobre CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML. Ele controla a cor, a fonte, o tamanho e a posição dos elementos, dando vida à estrutura da página.

Por que usar CSS externo?

Usar um arquivo CSS separado (`.css`) é considerado a melhor prática. Isso mantém o código HTML limpo e focado na estrutura, e permite que você aplique o mesmo estilo em várias páginas de um site, editando apenas um arquivo.
